Cool CLI commands for macos This is a list of amazing cli tools that you can brew or are present by default. yank yank is the yank tool for your cli. To use it simply pipe it after a command and it will allow you to yank a string in the output. For example, imagine you have a list of docker container running with complex names. Running docker ps | yank will allow you to capture the name of the docker container automatically....
